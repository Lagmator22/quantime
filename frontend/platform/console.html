New here? How to use this (30 seconds) ▾

QuanTime stress-tests trading matching engines - the software exchanges (NYSE, Binance) use to match buy/sell orders. Three steps:

  1. Pick an engine (left panel) and click ⚡ Use selected engine. Or upload your own - a .tar.gz with a Dockerfile + a POST /submit endpoint (see API & Docs).
  2. Click ▶ Launch benchmark. A distributed bot fleet floods it with orders; the metrics stream in live.
  3. Read the result. Lower latency = better · higher TPS & correctness = better.

Power moves: set Target rate / bot high → find the breaking point (latency explodes) · choose the replay profile → drive it with real market data · ★ Set as baseline then re-run → gate regressions.
